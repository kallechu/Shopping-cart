# Shopping-cart

## Overview

This is upcoming simple web application for managing shopping lists.
The idea for this app came, because me and my wife often forget our shopping list at home,
so now it can be stored online and is easily accessed from anywhere.

## Features
- Create, view, update and delete shopping lists
- Each list can contain multiple items with product and its count

## Technologies
- React (frontend)
- Node.js/Express (backend)
- MongoDB (database)
